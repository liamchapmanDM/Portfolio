2Veggie

Web-based Mobile Application. PHP, SQL, HTML5, CSS3, JavaScript.

Click here to visit app in browser.

The home page displays recipes in a database using PHP.

Using different SQL databses, users can add and remove favourites to their accounts.

Recipe pages are set in a tidy and professional layout.

I developed and designed 2Veggie as a web-based mobile application during my 3rd Year at University, where I also based it upon dissertation research into UX Design and future trends. This application was built to help users who were transitioning from eating meat to becoming vegetarian. I developed the app using PHP and using PHPmyAdmin in order to help organize the databases for the back end of the application. PHP was independently learnt and introduced to the application, in order to create a functional registration, log-in, favourites and search pages, as well as displaying data of the recipes. As a project I enjoyed working hard to learn a new language and working with the development side of the process as a problem solving task independently.
