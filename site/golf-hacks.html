Golf Hacks

- Streaming Media Website

Website based on Streaming Media using Databases. PHP, SQL, HTML, CSS, JavaScript.

Golf Hacks was a website I designed and developed during university, with a focus primarily on streaming videos using back-end databases with PHP and mySQL. Each video page is dynamically created using ID numbers, making the site easier to edit and change for multiple pages. This was my first full experience using PHP as a language, which I have been able to push on from. Behind the scenes, I have also developed an admin page, where the admin is able to separately log in, edit and upload videos to the site with a simple interface. While the sites designs could have been improved, I used this unit to vastly improve coding knowledge by learning about PHP, which has similarities with a range of other codes that are becoming more popular in industry.
