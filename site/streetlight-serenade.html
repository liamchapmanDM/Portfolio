Streetlight Serenade

- Band Website

Website for a local band to display music and information. HTML, CSS, JavaScript, Bootstrap.

I was asked to create a website for a local band during my second year at University with the ability to listen to songs and learn about them. This website is a simple one page layout with different sections of the page dedicated to different chapters. The site primarily uses HTML and CSS, while using twitter bootstrap to make it responsive for different screen sizes. The songs use a SoundCloud plugin so that they are embedded into the site and easy to share as well as access.
